# Modbus TCP 應用於統計數量

## 安裝 Install

python 3.11

```
pip install -r requirements.txt
```

## 執行 Run

在第一個命令提示字元執行
On cmd #1

``` 
python q1_server.py
```

在第二個命令提示字元執行
On cmd #2

```
python q1_client.py
```

在第三個命令提示字元執行
On cmd #3

```
python q1_line.py
```

步驟都執行後，作為示範，統計的數量會顯示在第二個命令提示字元
Once scripts is running, the ouput on cmd #2 demonstrates how to get the sum of the numbers.
